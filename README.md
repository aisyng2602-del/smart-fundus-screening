# smart-fundus-screening
An AI-powered healthcare solution for early-stage detection of Diabetic Retinopathy using deep learning on fundus images.



Overview
Smart Fundus Screening is an AI-based system designed to detect early signs of Diabetic Retinopathy (DR) using retinal (fundus) images. The goal is to assist in early diagnosis and prevent vision loss, especially in rural and low-resource areas.



 Features
-  Upload fundus (retinal) images  
-  AI-based DR detection (CNN Model)  
-  Classification: No DR, Mild, Moderate, Severe  
-  Fast and automated prediction  
-  Confidence score output

- **Database:** (Planned - Firebase/MySQL)  

---

##  How It Works
1. User uploads a retinal image  
2. Image is preprocessed (resize, enhancement)  
3. AI model analyzes the image  
4. System predicts DR stage  
5. Result is displayed with confidence score
